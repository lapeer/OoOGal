OoOGal

Authors: Drew Lapeer, Vicky Nelson, and Kesha Patel

Version 1.0.0

A simple framework to check what available data exists for a given galaxy. Currently, the package is able to query the MAST database for photometric data from JWST and HST. Future versions will expand the package into ALMA and SDSS as well.
